材质遮罩

角色的RGB遮罩 :

  • R = Metallic

  • G = Roughness

  • B = Subsurface Opacity

角色头发的RGB遮罩:

  • R = Hair Alpha

  • G = Specular/Roughness map

  • B = Anisotropic direction map

环境的RGB遮罩 t:

  • R = Metallic

  • G = Roughness

  • B = Ambient Occlusion
